# 武汉市自然资源和城乡建设局关于进一步优化建设工程规划许可管理的通知
文号：武自然资建规〔2026〕1号

## 一、总体要求
为持续优化营商环境，进一步提高行政审批效率，根据《中华人民共和国城乡规划法》、《武汉市城乡规划条例》等法律法规，结合我市实际，现就进一步优化建设工程规划许可管理有关事项通知如下。

## 二、简化审批流程
### （一）推行告知承诺制
对社会投资低风险工程建设项目，实行建设工程规划许可告知承诺制。建设单位在申请建设工程规划许可证时，书面承诺符合规划要求并提交必要材料的，审批部门直接作出审批决定。

### （二）合并办理事项
将建设工程规划设计方案审查与建设工程规划许可证核发合并办理。建设单位可同步申请，审批部门同步审查、同步发证。

## 三、精简申报材料
1. 取消施工图审查合格书作为核发建设工程规划许可证的前置条件。
2. 通过工程建设项目审批管理系统共享获取的材料，不再要求建设单位重复提交。

## 四、加强事中事后监管
审批部门应当在作出审批决定后2个月内，对被审批人的承诺内容是否属实进行核查。发现被审批人实际情况与承诺内容不符的，应当要求其限期整改；整改后仍不符合条件的，依法撤销行政许可决定。

## 五、施行时间
本通知自2026年3月1日起施行，有效期5年。
