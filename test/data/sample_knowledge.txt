# 工程测量规范 - 水准测量篇 (测试数据)

## 1. 适用范围
本规范适用于国家三、四等水准测量。

## 2. 精度要求
### 2.1 视线长度
*   三等水准测量：视线长度 ≤ 75m，前后视距差 ≤ 3.0m，累积差 ≤ 6.0m。
*   四等水准测量：视线长度 ≤ 100m，前后视距差 ≤ 5.0m，累积差 ≤ 10.0m。

### 2.2 读数限差
*   三等水准测量：黑红面读数差 ≤ 2.0mm，黑红面高差之差 ≤ 3.0mm。
*   四等水准测量：黑红面读数差 ≤ 3.0mm，黑红面高差之差 ≤ 5.0mm。

## 3. 观测顺序
1.  后视水准尺黑面
2.  前视水准尺黑面
3.  前视水准尺红面
4.  后视水准尺红面
